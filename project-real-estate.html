Mobdla — Real Estate CRM & Exchange App

Role: Lead Developer • Status: Mobile demo / screenshots

Project Overview

This platform enables property owners and buyers to discover, exchange and manage properties. It features AI-assisted matching, a conversational assistant for searches, and automated workflows for listings and offers.

Key Features

  • Mobile-first listing management and quick offers
  • Agent CRM and in-app messaging for leads
  • AI-assisted matching and recommendation engine
  • Push notifications, analytics and exportable reports

Tech Stack

React Native Node.js Postgres OpenAI / LLM Docker
